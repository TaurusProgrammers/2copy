Acerca de

2COPY Nace para dar solucion a problematicas en el ambito laboral y estudiantil

De que trata este proyecto

Manejamos una plataforma web en la cual podrás contar con una gama de servicios como la siguiente. 2COPY te brinda un pequeño espacio en la nube en el cual podrás subir tus archivos los cuales llegan ah un centro de copiado, en el que nuestro equipo de trabajo verifica los datos para luego

realizar el proceso de impresión y copiado el cual se hace un envió a la dirección solicitada de acuerdo a las especificaciones dadas por el usuario que solicito el servicio de una manera ágil y sencilla para la comodidad del usuario

2COPY se compromete con el usuario a brindar el mejor servicio posible tanto calidad como comodidad, tanto así que se ah asignado un historial de sus archivos dentro de la plataforma y una zona de controles para que tenga control sobre sus archivos mas importantes.

Nuestros servicios

Servicios que brinda nuestra plataforma para usuarios finales que desean ser parte de nuestra empresa

Panel de adminitracion

Se le brinda al usuario un panel el cual puede hacer uso para control de sus archivos

Actividades

El usuario tiene una serie de historiales y actividades recientes hecha en la plataforma de manera eficaz

Envio de archivo

Contiene un pequeño drive en los cuales el archivo puede subir archivos a la nube, los cuales puedes ser manipulados por el usuario cuando el lo disponga

Calidad de imagen

Nuestra plataforma garatiza la calidad de la imagen impresa a la hora de entrega a nuestro usuarios finales

Servicios de envio

Manejamos centros de copiado en la ciudad para la entrega eficaz en tu sitio de trabajo o instituto

Drive web

El usuario podra tener un tamaño asigando para el manejo de un portafolio de archivos en su plataforma admin
